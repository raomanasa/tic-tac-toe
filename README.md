# Tic-Tac-Toe!
## The code

It is a game software, for tic-tac-toe game.

## Setup-
Git Address
```
https://github.com/raomanasa/tic-tac-toe.git
```

## Setup Instructions
Initialize project with npm
```
$ npm init -y 
```
React installation 
```
$ npm i -S react react-dom 
```
